Getting Started

Welcome to the File Sharing System. Here's how to get started:

Uploading Files

To upload a file to the server:

  1. Navigate to the Upload page
  2. Select a file from your device
  3. Click the "Upload" button
  4. Wait for the progress bar to complete

Downloading Files

To download a file from the server:

  1. Go to List Files
  2. Find the file you want to download
  3. Click the "Download" button next to the file

Account Management

You need an account to access all features:

  • To create an account, visit the Register page
  • Already have an account? Login here

Admin Features

If you have admin privileges, you can:

  • Delete files from the server
  • View all user activities
  • Manage user accounts

Troubleshooting

Upload Issues

If you encounter problems during upload:

  • Check your internet connection
  • Ensure the file isn't too large (max size: 50MB)
  • Try using a different browser

Download Issues

If you have problems downloading files:

  • Make sure you're logged in
  • Check if the file still exists on the server
  • Contact an administrator if the issue persists
